# My Projects

This repository contains a set of static HTML/CSS pages, including a Netflix-style page and related project pages.

## Files

- HTML pages such as `netflix.html`, `login.html`, `games.html`, `players.html`, `teams.html`, `stats.html`, and `table.html`
- CSS stylesheets for each page

## Run Locally

1. Open the project folder in VS Code.
2. Open any HTML file in a browser (for example, `netflix.html`).

## Tech Stack

- HTML5
- CSS3
